# VMware Workstation Pro 17 Automation

This folder contains local lab automation for VMware Workstation Pro 17.

Scope:

- Clone a VM from a local template
- Update `.vmx` settings for CPU and memory
- Start and stop the VM with `vmrun`
- Hand off the guest OS to Ansible after boot

Recommended use:

- Local testing
- Fresher onboarding
- Script validation before moving to vSphere
